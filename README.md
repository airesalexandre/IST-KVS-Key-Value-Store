# IST-KVS: Key Value Store

Project for Operating Systems

## Project Overview

The IST-KVS (Key Value Store) is a distributed system built to store and manage data as key-value pairs using a client-server architecture. It supports commands to create, read, update, and delete entries, and includes features such as parallel processing, asynchronous backups, and clean shutdown of client connections. The system is designed to keep communication between clients and the server reliable and efficient.
